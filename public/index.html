About This Server

This server is designed to assist developers in generating payloads in order to claim drop game. Please follow the guidelines to avoid overloading the server.

Important Note for Developers

If you are a developer, please deploy your own instance of this server on Vercel or another platform. This will help keep the server stable and available for everyone. If everyone will use my instance, this can cause long outages.

How to Use the Server

Simply use post request to the /generate endpoint with json data to create your payload.

Refer to the api documentation for more detailed instructions.
